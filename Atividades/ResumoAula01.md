# Aula 01 - Lógica de Programação

## O que é um algoritmo
É uma instruções finitas para resolver um problema ou para fazer uma tarefa especifica

## Entrada, Processamento e Saída
 A entrada é a recepção de informações brutas do usuário
 Processamento é a manipulão ou a transformações de informações da entrada 
 e a saída é o resultado do processamento, ou seja é oque interessa pro usuario 

## Exemplo do dia a dia

Exemplo: Fazer um Login no Email

1 - Ligar o Computador
2 - Verficar se tem acesso a internet
3 - Entrar no email
4 - Colocar o email e a senha correta
5 - Entrar no Email

## 5 palavras-chave

Algoritmo:
Instruções finitas

Entrada:
Entrada de Dados

Processamento:
Manipulação dos dados da entrada

Saída:
Resultado dos dados que chega no usuário

Lógica:
Organização do pensamento
